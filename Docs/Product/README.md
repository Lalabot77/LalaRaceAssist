# Product Surfaces

Lala Race Assist is organized around the surfaces the driver or support crew interacts with first.

- [Driver Dash](Driver_Dash.md) — tactical in-car support.
- [Strategy Dash](Strategy_Dash.md) — race-management and planning support.
- [Plugin UI](Plugin_UI.md) — setup, profiles, planning, bindings, settings, and review.
- [Overlays](Overlays.md) — temporary supporting surfaces.

These pages describe what the user sees. Technical ownership lives in [Subsystems](../Subsystems/README.md).
