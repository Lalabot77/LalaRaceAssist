# Standing Start Assist

## What it does

Standing Start Assist supports standing-start setup, clutch-drop/start capture, saved summaries, launch trace review, and post-start learning.

## First-use setup

Configure start settings, capture multiple starts before tuning, and review summaries/traces for repeatable patterns rather than one-off mistakes.

## What to trust / review

Trust it as review and setup support. If it feels wrong, compare several clean starts, check settings, and separate driver execution mistakes from repeatable setup problems.

## What it does not own

Standing Start Assist does not rename Launch Mode internals, launch traces, SimHub action/export names, settings fields, dashboard contracts, or plugin file names such as LaunchPlugin.dll.
