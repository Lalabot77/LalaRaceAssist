# Standing Start Assist

Standing Start Assist is for standing-start work: setup, clutch-drop/start capture, saved summaries, launch trace review, and post-start analysis. It is not about launching the plugin; it is about understanding and improving the first seconds of a standing start.

Configure the start settings before you judge the results. Then capture several starts before tuning aggressively. One bad launch can be a driver mistake, a track condition, or a timing anomaly; repeatable patterns across multiple clean starts are much more useful.

Use the review surface to compare what actually happened against what you intended. Look for consistent clutch bog, wheel slip, throttle behaviour, or timing patterns. If you only react to the most recent start, you are likely to over-correct.

Trust Standing Start Assist as a review and setup aid when you have comparable clean captures. Question it when the run was messy, the settings changed between attempts, the trace is incomplete, or you are comparing different cars/conditions.
