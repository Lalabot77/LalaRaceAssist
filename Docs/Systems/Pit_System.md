# Pit System

## What it does

Pit System helps with pit timing/loss, pit entry, pit box, commands, fuel/tyre control, debriefs, and track markers.

## First-use setup

Validate pit markers and pit loss. Learn pit loss with a clean limiter-speed drive-through, review pit entry buffer/decel per car, and remember that pit service regulations are manual Strategy assumptions.

## What to trust / review

Trust pit guidance most after clean marker/loss validation. If it feels wrong, review pit markers, pit-loss learning, pit-entry decel/buffer, service assumptions, and fuel/tyre command feedback.

## What it does not own

Pit System does not own Strategy race assumptions beyond consuming them, Traffic/Race Awareness selection, or dashboard layout.
