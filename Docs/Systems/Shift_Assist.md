# Shift Assist

Shift Assist is a cueing and learning system for cars where shift timing benefits from guidance. It helps you learn and repeat useful shift points; it does not drive the car for you or change gearbox behaviour.

Enable it only for cars where it makes sense. The best learning data comes from clean full-throttle pulls, not messy laps where traffic, lifts, damage, or mistakes distort the sample. Once the learned values look stable, lock the gear values you trust so they do not drift from poor future evidence.

Treat the cues as assistance, not an order. If the cue feels slightly early or late, review the learned gear value and the conditions that produced it before assuming the system is wrong. Audio/output setup also matters: a good learned value is still hard to use if the cue is delayed, too quiet, or routed to the wrong device.

Trust Shift Assist after stable learning and review. Question it after setup changes, different gearing, weak samples, partial-throttle learning, unusual track conditions, or any car where the ideal shift behaviour is not consistent.
