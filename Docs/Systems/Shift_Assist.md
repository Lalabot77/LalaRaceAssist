# Shift Assist

## What it does

Shift Assist provides shift cueing, learning, audio/review support, and per-car guidance.

## First-use setup

Enable it only for cars where it is useful. Learn during clean full-throttle pulls, review learned gear values, and lock stable values once validated.

## What to trust / review

Trust cues after stable learning and review. If cues feel wrong, check gear-specific learned values, lock state, audio/output setup, and whether the car/track conditions changed.

## What it does not own

Shift Assist assists the driver; it does not shift the car, change gearbox behavior, or own Strategy/Pit/Race Awareness logic.
